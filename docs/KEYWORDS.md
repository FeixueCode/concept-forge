# Skill 描述与关键词

## Skill 名称

Concept Forge

## 中文名建议

概念锻造器

## 一句话描述

把一个随口提出的问题或概念，转成一句直接解释、一份 Quarkdown 知识母稿、一个互动 HTML 页面和一份 Slidev 演示稿源码。

## 长描述

Concept Forge 用于处理用户日常遇到的碎片化知识：陌生名词、新工具、技术概念、方法论、缩写、产品、工作流，以及“X 是什么 / X 和 Y 有什么区别”这类问题。它会先像 hv-analysis 一样做轻量研究和判断，再把内容整理成结构化知识记录，输出 Quarkdown 母稿、互动 HTML 和 Slidev 演示稿源码，同时维护本地索引，方便下次遇到同类问题时直接复用。

## 触发场景

- 用户问某个概念是什么。
- 用户问两个概念有什么区别。
- 用户给出一个陌生关键词。
- 用户希望把零散知识整理成知识库。
- 用户希望得到可阅读、可复习、可讲解的材料。
- 用户希望下次再问时直接复用旧结果。
